# Конвертер регистров 

Учебный проект.

Конвертер регистров позволяет менять:
ВЕРХНИЙ регистр на НИЖНИЙ,
НИЖНИЙ на ВЕРХНИЙ,
одновременно ВЕРХНИЙ на НИЖНИЙ, а НИЖНИЙ на ВЕРХНИЙ,
регистр первой буквы слова на ВЕРХНИЙ, а остальные буквы делать НИЖНЕГО регистра.

Поддерживается замена текста только для английский букв из таблицы ASCII.

## Командная строка

changecase -key source_file destinstion_filename

Поддерживаемые ключи:
- "-L" ВЕРХНИЙ - нижний
- "-U" нижний - ВЕРХНИЙ
- "-G" ВЕРХНИЙ - нижний, нижний - ВЕРХНИЙ
- "-T" первая буква - ВЕРХНИЙ, другие буквы - нижний
